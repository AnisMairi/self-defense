# ASSETS_PLAN — Photos & médias (libre de droit + logo)

## Dossiers
- `/src/assets/logos/` : logos fournis
- `/src/assets/images/` : photos libres de droit (webp)
- `/src/assets/icons/` : icônes / pictos éventuels

## Règles techniques
- Format : **.webp**
- Largeur :
  - Hero : 1600–2200px
  - Cards : 1200px
- Optimisation : compression + lazy loading
- Alt text : descriptif + sobre (“Entraînement krav maga en salle”)

## Style photo (mix)
### Sport / action
- Krav‑Maga training
- Boxing coaching / heavy bag
- Self‑defense drills
- Group class in gym (salle sombre)

### Institutionnel / formation
- Formateur devant un groupe
- Séminaire / salle de formation
- Échanges / briefing
- Posture professionnelle
